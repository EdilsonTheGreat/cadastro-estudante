# cadastro_aula8

O presente projeto consiste no desenvolvimento de uma aplicação móvel, cujo principal objetivo é permitir o registo, autenticação e visualização de disciplinas associadas a um estudante.

A aplicação foi concebida no contexto académico, com foco na aplicação de conceitos fundamentais de Programação Orientada a Objetos (POO) e da arquitetura MVVM (Model-View-ViewModel), promovendo a separação de responsabilidades e a organização do código.
